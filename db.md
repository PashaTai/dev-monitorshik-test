- id (PRIMARY KEY AUTOINCREMENT)
- source (TEXT: 'telegram' / 'vk')
- source_comment_id (TEXT: уникальный ID, для дедупликации)
- group_channel_name (TEXT)
- author_name (TEXT)
- author_id (TEXT)
- author_username (TEXT, nullable)
- comment_text (TEXT)
- has_media (INTEGER: 0/1)
- media_type (TEXT: 'photo'/'video'/'sticker'/'voice'/'large_video', nullable)
- post_url (TEXT)
- post_published_at (TIMESTAMP)
- comment_url (TEXT)
- sentiment (TEXT: nullable, потом 'positive'/'negative'/'neutral')
- sentiment_score (REAL: nullable)
- comment_published_at (TIMESTAMP)
- parsed_at (TIMESTAMP)
- processed (INTEGER: 0/1)

Индексы для быстрых запросов:

По source_comment_id + source (дедупликация)
По published_at (фильтры по датам)
По processed (поиск необработанных для sentiment)
